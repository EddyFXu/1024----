# 1024 图片爬虫工具 (v1.0.0)

这是一个专为图片论坛设计的批量下载工具，支持自动翻页、智能探索、分辨率过滤和自定义重命名。

## ✨ 功能特性

*   **图形化界面**：操作简单直观，无需编写代码。
*   **智能导航**：
    *   **自动下一主题**：按顺序下载下一个帖子。
    *   **自动上一主题**：按顺序下载上一个帖子。
    *   **自由探索**：自动分析页面链接，智能跳转。
*   **精准过滤**：
    *   **分辨率筛选**：设定最小宽高（如 1920x1080），自动剔除低清缩略图或广告图。
    *   **格式筛选**：支持 jpg, png, webp, gif, svg 等常见格式。
*   **高级重命名**：支持使用帖子标题、发表日期、序号等信息自定义文件和文件夹命名。
*   **防封机制**：内置随机延迟和 User-Agent 伪装，降低被服务器封锁的风险。
*   **实时预览**：下载过程中实时显示任务状态和图片预览。
*   **断点续传**：自动记录最后成功下载的地址，意外中断后可快速恢复。
*   **实时监控**：显示实时下载带宽和速率。

## 🚀 快速开始

### 1. 环境准备

确保你已经安装了 Python (建议 3.8+)。
在项目目录下打开终端，安装所需依赖：

```bash
pip install -r requirements.txt
```

### 2. 运行程序

*   **方式一（推荐）**：双击运行目录下的 `run.bat` 脚本。
*   **方式二**：在终端运行 `python main.py`。

## 📖 使用指南

### 1. 基础设置
*   **目标地址**：在输入框中粘贴你想要开始下载的帖子 URL（例如：`https://.../read.php?tid=xxxxx`）。
*   **探索模式**：
    *   通常选择 **"自动下一主题"** 即可连续下载后续帖子。
    *   如果你是从最新的帖子开始看，可以选择 **"自动上一主题"**。

### 2. 图片过滤
*   **最小分辨率**：下拉选择或手动输入（格式如 `1000x1000`）。设置后，程序会自动删除尺寸小于该值的图片（非常适合过滤掉无关的头像、表情包和广告）。
*   **图片格式**：默认全选，你可以根据需要取消勾选不需要的格式。

### 3. 保存与命名 (重点)
你可以完全自定义图片保存的路径和文件名。

**支持的命名占位符：**
*   `{filename}`: 原始文件名
*   `{no.10001}`: 自增序号 (从 10001 开始)
*   `{origin_serial}`: 原始链接中的序号（如有）
*   `{page.title}`: 帖子标题
*   `{page.date}`: 帖子发表日期 (YYYY-MM-DD)
*   `{YYYY}`: 当前年份
*   `{MM}`: 当前月份
*   `{DD}`: 当前日期
*   `{HH}`: 当前小时
*   `{mm}`: 当前分钟
*   `{ss}`: 当前秒数

**示例：**
*   按日期和标题归档：`D:\Downloads\{YYYY-MM-DD}\{page.title}\{filename}`
*   简单重命名：`{page.title}_{no.001}.jpg`

## 🛠️ 编译为 EXE

如果你想将程序打包为独立的可执行文件，请运行根目录下的 `build.bat` 脚本。

```bash
.\build.bat
```

编译完成后，可执行文件将生成在 `dist` 目录下。
注意：编译前请确保安装了 `pyinstaller` (`pip install pyinstaller`) 和 `Pillow` (`pip install Pillow`)。
